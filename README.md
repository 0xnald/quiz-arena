# QuizArena

Interactive live quiz platform for communities, classrooms, and events.

Inspired by GenLayer Community · Built by Amie

## Features

- Create quizzes with up to 50 questions (4 options each)
- Host live sessions with up to 500 players
- Speed-based scoring (10-100 points per question)
- 15-second timer per question
- Real-time leaderboard with podium display
- Participants join via game PIN — no signup needed
- Beautiful dark purple UI

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm run build
```

Or connect your GitHub repo to Vercel for automatic deployments.

## Tech Stack

- React 18
- Vite
- Vanilla CSS-in-JS (no dependencies)
